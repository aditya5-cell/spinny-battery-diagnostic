# Spinny Battery Diagnostic System v2

AI-powered battery diagnostic tool for Spinny's technician portal.

## Live Demo
https://tourmaline-rabanadas-156360.netlify.app

## Features
- Arrhenius temperature acceleration modelling
- Non-linear battery aging curves
- OCV–SOC 11-point lookup table
- Weibull survival analysis for failure prediction
- City-specific climate factors for 15 Indian cities
- Gemini AI integration for technician reports

## Tech Stack
React.js, Gemini AI API
